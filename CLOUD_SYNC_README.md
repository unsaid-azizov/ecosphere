### ✅ **Что синхронизируется с базой данных PostgreSQL:**

#### 🛒 **Корзина**
- ✅ **Автосинхронизация** с базой данных при каждом изменении
- ✅ **Восстановление** корзины при входе в аккаунт
- ✅ **Локальное хранение** для неавторизованных пользователей
- ✅ **Автоочистка** корзины после успешного заказа
- ✅ **Реальное время** - изменения сразу сохраняются в БД

#### ❤️ **Избранные товары**
- ✅ **Моментальная синхронизация** при добавлении/удалении
- ✅ **Восстановление** из базы данных при входе
- ✅ **Быстрый доступ** через личный кабинет
- ✅ **Кнопки избранного** на всех карточках товаров

#### 📦 **Заказы**
- ✅ **Полная история заказов** с детальной информацией
- ✅ **Статусы заказов** (Ожидает, Подтвержден, В обработке, Отправлен, Доставлен)
- ✅ **Контактная информация** и адрес доставки
- ✅ **Детализация товаров** в каждом заказе
- ✅ **Оформление заказов** через диалоговое окно

#### 👤 **Пользовательские данные**
- ✅ **Профили пользователей** с типами (физ.лицо, ИП, ООО)
- ✅ **Персональные скидки** по типу пользователя
- ✅ **VIP статусы** и специальные цены
- ✅ **Безопасная авторизация** через NextAuth.js

## 🌐 **Логика синхронизации**

### **Авторизованные пользователи:**
- 📡 Все действия **мгновенно сохраняются** в PostgreSQL
- 🔄 При входе все данные **восстанавливаются** из облака
- ⚡ **Реальное время** - никаких потерь данных

### **Неавторизованные пользователи:**
- 💾 Корзина сохраняется **локально** в браузере
- 🔐 При авторизации данные **переносятся** в облако
- 🚫 Избранное и заказы **требуют авторизации**

- 📈 Количество заказов из базы данных

## 🚀 **API Endpoints для синхронизации**

### Корзина
```
GET    /api/cart - Загрузить корзину из БД
POST   /api/cart - Добавить товар в корзину
PUT    /api/cart - Обновить количество
DELETE /api/cart - Удалить товар/очистить корзину
```

### Избранные товары  
```
GET    /api/favorites - Получить избранное
POST   /api/favorites - Добавить в избранное
DELETE /api/favorites - Удалить из избранного
```

### Заказы
```
GET    /api/orders - История заказов
POST   /api/orders - Создать новый заказ (с автоочисткой корзины)
```

### Скидки и цены
```
GET    /api/discounts - Получить персональные скидки и цены
POST   /api/discounts - Установить специальную цену (админ)
```

## 🎯 **Что происходит при действиях пользователя:**

### ➕ **Добавление товара в корзину:**
1. Товар добавляется в локальное состояние (мгновенно)
2. Отправляется запрос в `/api/cart` (синхронизация)
3. Сохраняется в PostgreSQL
4. При следующем входе восстанавливается из БД

### ❤️ **Добавление в избранное:**
1. Кнопка сердечка меняется мгновенно
2. Запрос отправляется в `/api/favorites`
3. Сохраняется в БД с связью на пользователя
4. Доступно на странице избранного

### 🛍️ **Оформление заказа:**
1. Открывается диалог с формой
2. Подтягиваются данные профиля (email автоматически)
3. При отправке создается запись в таблице `orders`
4. Добавляются все товары в `order_items`
5. **Корзина автоматически очищается**
6. Пользователь перенаправляется на страницу заказов

### **Обработка ошибок:**
- При ошибке сети данные остаются локально

### **Производительность:**
- Минимизированные запросы к API
- Кэширование данных в Context
- Оптимистичные обновления UI

### **Безопасность:**
- Все API защищены NextAuth сессиями
- Пользователи видят только свои данные
- CSRF защита из коробки
