# ЭкоСфера - Backend Documentation

## Реализованная функциональность

### ✅ Система аутентификации
- **Регистрация** с 3 типами пользователей:
  - 🟢 **Физическое лицо** - личные покупки
  - 🔵 **ИП** - индивидуальный предприниматель (10% скидка)
  - 🟣 **ООО** - юридическое лицо (15% скидка)
- **Авторизация** через NextAuth.js с credentials provider
- **Личный кабинет** с информацией о пользователе
- **Защищенные маршруты** и сессии

### ✅ База данных PostgreSQL
- **Полная схема** с 8 таблицами
- **Отношения** между сущностями
- **Миграции** через Prisma
- **Типизация** на TypeScript

### ✅ Система скидок и специальных цен
- **Иерархия скидок**:
  1. Специальная цена (приоритет #1)
  2. VIP скидка 15% (приоритет #2)  
  3. Скидка по типу: ИП 10%, ООО 15% (приоритет #3)
  4. Персональная скидка (приоритет #4)
  5. Акционная скидка на товар (приоритет #5)
- **Динамическое** вычисление цен
- **Визуальное** отображение скидок

### ✅ Избранные товары
- **API** для добавления/удаления из избранного
- **Context** для управления состоянием
- **UI компоненты** с анимациями
- **Страница** избранных товаров

## API Endpoints

### Аутентификация
```
POST /api/auth/register - Регистрация пользователя
POST /api/auth/[...nextauth] - NextAuth.js endpoints
GET  /api/auth/session - Получить сессию
```

### Скидки и цены
```
GET  /api/discounts - Получить скидки для пользователя
POST /api/discounts - Установить специальную цену (админ)
```

### Избранные товары
```
GET    /api/favorites - Получить список избранных
POST   /api/favorites - Добавить в избранное
DELETE /api/favorites?productId=X - Удалить из избранного
```

### Корзина (API готов, нужна интеграция с UI)
```
GET    /api/cart - Получить корзину пользователя
POST   /api/cart - Добавить товар в корзину
PUT    /api/cart - Обновить количество товара
DELETE /api/cart?productId=X - Удалить товар / очистить корзину
```

## Страницы

- `/` - Главная страница
- `/catalog` - Каталог товаров с системой цен
- `/dashboard` - Личный кабинет пользователя
- `/favorites` - Избранные товары
- `/cart` - Корзина
- `/orders` - История заказов

## Тестирование

### 1. Регистрация
1. Откройте `http://localhost:3000`
2. Нажмите "Зарегистрироваться"
3. Выберите тип пользователя (попробуйте ИП или ООО для скидок)
4. Заполните форму и зарегистрируйтесь

### 2. Авторизация
1. Нажмите "Войти" 
2. Введите email и пароль
3. После входа увидите ваше имя в навигации

### 3. Система скидок
1. Перейдите в каталог товаров
2. Увидите **персональные цены** с учетом типа пользователя
3. Скидки отображаются бейджами рядом с ценой

### 4. Избранные товары
1. Нажмите ❤️ на карточке товара
2. Товар добавится в избранное
3. Перейдите на страницу "Избранное" через личный кабинет

### 5. Личный кабинет
1. Нажмите на имя пользователя в навигации
2. Выберите "Личный кабинет"
3. Увидите статистику и быстрые действия

## База данных

### Подключение
```
DATABASE_URL="postgresql://ecosphere_user:ecopassword@localhost:5432/ecosphere"
```

### Основные таблицы
- `users` - Пользователи с типами и скидками
- `orders` - Заказы и их статусы
- `order_items` - Товары в заказах
- `favorites` - Избранные товары
- `user_special_prices` - Специальные цены
- `product_discounts` - Скидки на товары
- `product_inventory` - Остатки товаров

## Технологический стек

- **Framework**: Next.js 14 + TypeScript
- **Database**: PostgreSQL + Prisma ORM
- **Auth**: NextAuth.js
- **UI**: Tailwind CSS + shadcn/ui
- **State**: React Context API
